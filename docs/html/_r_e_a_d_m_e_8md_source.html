9 This is a self-contained Linux userspace C interface for the Invensense MPU series of IMUs and should work with the MPU6050, MPU6500, MPU9150, and MPU9250. I only have an MPU9250 to test on right now so please
10 
11 It shares the code with, and is primarily developed on, the BeagleBone Black Robotics Cape and Beaglebone Blue. However it has also been tested on the Raspberry Pi and Nvidia TX1. The API and source files here are borrowed from the Robotics Cape library.project.
12 
13 Building
14 --------
15 
16 You will likely need to change the I2C bus in the example programs to match your connection. The examples default to using I2C bus 2 to match the Robotics Cape and Beaglebone Blue.
17 
18 
19  ~$ git clone https://github.com/StrawsonDesign/rc_mpu.git
20  ~$ cd rc_mpu
21  ~/rc_mpu$ make
