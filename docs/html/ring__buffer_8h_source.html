1 /*******************************************************************************
2 * Ring Buffer
3 *
4 * Ring buffers are FIFO (first in first out) buffers of fixed length which
5 * efficiently boot out the oldest value when full. They are particularly well
6 * suited for storing the last n values in a discrete time filter.
7 *
8 * The user creates their own instance of a buffer and passes a pointer to the
9 * these ring_buf functions to perform normal operations.
10 *
11 * @ int rc_alloc_ringbuf(rc_ringbuf_t* buf, int size)
12 *
13 * Allocates memory for a ring buffer and initializes an rc_ringbuf_t struct.
14 * If ring buffer b is already the right size then it is left untouched.
15 * Otherwise any existing memory allocated for buf is free'd to avoid memory
16 * leaks and new memory is allocated. Returns 0 on success or -1 on failure.
17 *
18 * @ rc_ringbuf_t rc_empty_ringbuf()
19 *
20 * Returns an rc_ringbuf_t struct which is completely zero'd out with no memory
21 * allocated for it. This is useful for declaring new ring buffers since structs
22 * declared inside of functions are not necessarily zero'd out which can cause
23 * the struct to contain problematic contents leading to segfaults. New ring
24 * buffers should be initialized with this before calling rc_alloc_ringbuf.
25 *
26 * @ int rc_free_ringbuf(rc_ringbuf_t* buf)
27 *
28 * Frees the memory allocated for buffer buf. Also set the initialized flag to 0
29 * so other functions don't try to access unallocated memory.
30 * Returns 0 on success or -1 on failure.
31 *
32 * @ int rc_reset_ringbuf(rc_ringbuf_t* buf)
33 *
34 * Sets all values in the buffer to 0 and sets the buffer index back to 0.
35 * Returns 0 on success or -1 on failure.
36 *
37 * @ int rc_insert_new_ringbuf_value(rc_ringbuf_t* buf, float val)
38 *
39 * Puts a new float into the ring buffer and updates the index accordingly.
40 * If the buffer was full then the oldest value in the buffer is automatically
41 * removed. Returns 0 on success or -1 on failure.
42 *
43 * @ float rc_get_ringbuf_value(rc_ringbuf_t* buf, int pos)
44 *
45 * Returns the float which is 'pos' steps behind the last value added to the
46 * buffer. If 'position' is given as 0 then the most recent value is returned.
47 * Position 'pos' obviously can't be larger than the buffer size minus 1.
48 * Prints an error message and return -1.0f on error.
49 *
50 * @ float rc_std_dev_ringbuf(rc_ringbuf_t buf)
51 *
52 * Returns the standard deviation of the values in the ring buffer.
53 *******************************************************************************/
